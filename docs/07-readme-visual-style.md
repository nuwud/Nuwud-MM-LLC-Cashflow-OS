# 🎨 README Visual Style Guide

Nuwud docs should feel alive, scannable, and useful.

Some teams prefer plain, dry READMEs. That is fine for them.

For Nuwud Multimedia LLC, emojis and visual cues are part of the operating interface. They help Patrick scan complex systems quickly, remember categories, and feel momentum instead of staring at gray walls of text.

## 🧭 Core Principle

Use emojis as **wayfinding markers**, not decoration spam.

Each icon should help the reader instantly understand the type of section, task, risk, or action.

## ✅ Good Uses

| Emoji Type | Use |
| --- | --- |
| 💸 Money | cashflow, invoices, payments, revenue |
| 🔁 Loop | repeatable operating rhythms |
| 🎯 Target | goals, focus, priorities |
| 🧲 Capture | prospecting, lead capture, attraction |
| 💬 Conversation | outreach, follow-ups, replies |
| 🔍 Diagnosis | audits, investigation, research |
| 🧾 Offer / Invoice | paid audit, proposal, payment request |
| 🤝 Relationship | clients, partnerships, affiliate alignment |
| ⚙️ Automation | n8n, workflows, systems |
| 📊 Dashboard | metrics, trackers, databases |
| 📣 Content | LinkedIn, X, authority posts |
| ⚠️ Risk | blockers, warnings, guardrails |
| 🧠 Strategy | founder judgment, architecture, AI-native thinking |
| 🧱 Build | implementation, infrastructure, reusable systems |
| ✅ Done | completion, acceptance criteria |

## 🚫 Avoid

- random emojis that do not clarify meaning
- five emojis in a row for the same idea
- emojis inside every sentence
- decorative noise that slows scanning
- replacing precise language with cute symbols

## 🧩 Section Pattern

Preferred structure:

```md
## 🎯 Section Title

Short purpose statement.

| Thing | Value |
| --- | --- |
| Example | Example |
```

## 🚦 Status Markers

Use consistent status cues:

- 🟢 Active / moving
- 🟡 Needs attention
- 🔴 Blocked / urgent
- ⚫ Archived / inactive
- ✅ Done
- ⏳ Waiting
- ⚠️ Risk

## 💰 Cashflow Markers

Use these when money is involved:

- 💸 Cashflow
- 💰 Payment collected
- 🧾 Invoice / audit offer
- 🔐 Deposit required
- 🚫 Work blocked until payment

## 📣 Content Markers

Use these for content systems:

- 📣 Published / post
- ✍️ Draft
- 🧲 Lead magnet / attraction
- 🤝 Affiliate / partner mention
- ⚠️ Disclosure needed

## ⚙️ Automation Markers

Use these for n8n/workflows:

- ⚙️ Automation
- 🔔 Reminder / notification
- 🧠 AI-assisted step
- 🧍 Human review required
- 🛑 Do not auto-send

## ✍️ Writing Tone

Nuwud operating docs should be:

- sharp
- direct
- visually scannable
- cashflow-aware
- practical
- slightly energetic
- not corporate beige

## 🧠 Why This Matters

The Cashflow OS is not just documentation. It is a behavior-shaping interface.

If the page feels dead, Patrick is less likely to use it.

If the page gives quick visual cues, the system becomes easier to run daily.

The goal is not to impress documentation purists.

The goal is to make the machine usable.
