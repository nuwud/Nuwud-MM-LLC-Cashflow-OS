# Daily and Weekly Cashflow Rhythm

The rhythm exists to stop Nuwud from disappearing into builder mode before money moves.

## Daily command sequence

Run this before deep work.

### 1. Money check

Answer:

- What money is expected?
- What invoices need to be sent?
- What payment follow-up is needed?
- What work is blocked until payment?

### 2. Pipeline check

Answer:

- Who needs follow-up today?
- Who has no next action?
- Who can realistically pay within 14 days?
- Which prospect should receive the paid audit offer?

### 3. Sales actions first

Minimum daily sales motion:

- Send 3 follow-ups, or
- Start 3 qualified conversations, or
- Send 1 invoice/proposal plus 2 follow-ups

Builder work starts after this.

## Daily scoreboard

| Metric | Target |
| --- | ---: |
| Follow-ups sent | 3–5 |
| New prospects captured | 3–10 |
| Conversations started/restarted | 2–5 |
| Paid audit offers made | 1 if opportunity exists |
| Invoices/proposals moved | 1 if opportunity exists |
| Public authority post | 1 optional |

## Weekly CEO pass

Run every Friday.

### Review

- Cash collected
- Invoices sent
- Open proposals
- Warm opportunities
- Lost / stalled opportunities
- Content published
- Affiliate clicks / conversions if available
- Delivery bottlenecks
- Scope creep risks

### Decide

- Which 3 prospects are most likely to pay next week?
- Which offer is easiest to sell now?
- What needs invoicing?
- What needs to be paused until payment?
- What content should be created from actual client problems?

## Weekly targets

| Metric | Minimum | Strong week |
| --- | ---: | ---: |
| Prospects added | 20 | 50 |
| Follow-ups | 15 | 30 |
| Conversations | 10 | 20 |
| Audit offers | 3 | 5 |
| Invoices sent | 1 | 3 |
| Paid audits closed | 1 | 3 |
| Authority posts | 3 | 5 |
| Affiliate/tool posts | 1 | 3 |

## Saturday cleanup

Optional, but useful:

- clean tracker
- archive dead leads
- turn client lessons into content ideas
- prep Monday follow-ups
- update affiliate/resource list

## Sunday reset

Do not use Sunday to panic-build.

Use it to pick the next three money moves.
