## Required fields

| Field | Type | Purpose |
| --- | --- | --- |
| Prospect Name | Text | Person or primary contact |
| Business Name | Text | Company / brand |
| Website | URL | Site to review |
| Platform | Select | Shopify, WordPress, WooCommerce, Custom, Unknown |
| Source | Select | Warm, Past Client, Referral, X, LinkedIn, Local, Research, Other |
| Persona | Select | Shopify Operator, Local Premium Service, Subscription Brand, Agency, Founder, Other |
| Pain Observed | Text | What seems broken or leaking money |
| Offer Angle | Select | Audit, Sprint, Buildout, Retainer, Affiliate Resource, Referral |
| Status | Select | Captured, Researched, Contacted, Replied, Discovery, Audit Offered, Invoice Sent, Paid, Delivered, Proposal Sent, Won, Lost, Nurture |
| Priority | Select | A, B, C |
| Potential Value | Number | Estimated opportunity size |
| Next Action | Text | The exact next move |
| Next Action Date | Date | When to move it |
| Last Touch | Date | Last contact date |
| Notes | Text | Context |
| Affiliate Fit | Select | None, Hosting, CRM, Automation, SEO, Ecommerce, AI, Analytics |
| Content Angle | Text | Possible public lesson/post |

## Status definitions

| Status | Meaning |
| --- | --- |
| Captured | Added but not researched |
| Researched | Basic pain / fit identified |
| Contacted | Initial message sent |
| Replied | They responded |
| Discovery | Conversation about pain is happening |
| Audit Offered | Paid diagnostic has been offered |
| Invoice Sent | Invoice/payment link sent |
| Paid | Money collected |
| Delivered | Audit/deliverable completed |
| Proposal Sent | Implementation proposal sent |
| Won | Implementation or retainer closed |
| Lost | Not a fit or declined |
| Nurture | Keep warm, no current deal |

## Priority rules

### A priority

- can likely pay in 14 days
- clear pain
- real business
- decision-maker access
- strong fit for paid audit or implementation

### B priority

- good fit but slower timing
- needs nurturing
- unclear budget but credible business

### C priority

- weak fit
- early-stage
- low urgency
- mostly content/resource value

## Next action examples

Good:

- Send $1,500 investigation invoice.
- Follow up about missing materials.
- Offer paid Revenue Systems Audit.
- Ask if they want the leak map before rebuilding.
- Send calendar link for 20-minute discovery call.

Bad:

- Check in sometime.
- Think about proposal.
- Research more.
- Wait.

## Weekly pipeline hygiene

Every Friday:

- no A-priority prospect may have an empty next action
- no replied lead may sit untouched for more than 3 business days
- every invoice has a follow-up date
- every paid audit has a delivery date
- every delivered audit has an implementation proposal status
